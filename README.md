# How to run the project

1. Create a `.env` file with the correct values for the API keys. 
There are two required variables:
- `VITE_EXCHANGE_RATE_API_KEY=xxx`
- `VITE_EXCHANGE_RATE_API_URL=xxx`

2. Install dependencies
```bash
npm install
```

3. Run the project
```bash
npm run dev
```

# How to run tests

```bash
npm run test
```
